# Portable-Personal-Information-Detector
>本研究將提出個人資料防洩密裝置之設計，並應用開源軟硬體進行實作，對使用者使用網路時的封包進行個人資料的攔截與使用正規表示法對封包進行解析，讓使用者可以瞭解自己在使用網路服務的情況，達到提升使用者對資訊安全的安全意識，並減少個資外洩之風險。
